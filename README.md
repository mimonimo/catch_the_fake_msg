# 🕵️‍♂️ 가짜 메시지를 잡아라! (Catch the Fake Message)

**"수상한 링크는 꾹 참아요! 내 정보는 소중하니까요!"**

2026년 5월 5일 어린이날, **창의융합교육원**에서 진행하는 창의융합축제 교육용 웹 프로젝트입니다. 초등학생 친구들이 스팸 문자와 피싱 메시지를 구별하는 방법을 쉽고 재미있게 배울 수 있도록 퀴즈 형식으로 제작되었습니다.

---

## 📅 프로젝트 배경
- **행사명**: 5월 5일 창의융합축제 (어린이날 행사)
- **장소**: 창의융합교육원
- **대상**: 초등학생 및 학부모
- **컨셉**: '거짓말 메시지'를 찾아라! (스팸 문자 대응 교육)

## 🎮 주요 학습 내용 (꼭 기억하기!)
아이들이 게임을 통해 자연스럽게 습득할 수 있는 3가지 핵심 규칙입니다.

1. **모르는 링크는 꾹 참아요!** 🎁
   - "공짜 선물", "게임 아이템" 유혹에 속지 않고 수상한 링크(URL) 클릭하지 않기.
2. **내 정보는 함부로 주지 않아요!** 🤫
   - 아는 어른처럼 접근해도 이름, 전화번호, 사진 등 소중한 개인정보 보내지 않기.
3. **이상한 연락은 부모님께 먼저 물어봐요!** 🚨
   - 재촉하는 문자나 돈을 준다는 문자를 받으면 반드시 부모님께 확인받기.

## 🛠 사용 기술 (Tech Stack)
초보자도 쉽게 이해하고 수정할 수 있는 가장 기본적인 웹 기술로 제작되었습니다.
- **HTML5**: 웹사이트의 구조와 내용
- **CSS3**: 아기자기하고 보기 편한 디자인
- **JavaScript**: 퀴즈 정답 확인 및 인터랙션 기능

## 🚀 실행 방법
1. 이 저장소를 `clone` 하거나 파일을 다운로드합니다.
2. 폴더 내의 `index.html` 파일을 더블 클릭합니다.
3. 웹 브라우저(Chrome, Edge 등)에서 바로 게임을 시작할 수 있습니다!
